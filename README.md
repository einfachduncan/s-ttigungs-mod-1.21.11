# Sättigungs Mod für Minecraft 1.21.11

Eine Fabric-Mod für Minecraft 1.21.11, die es dir ermöglicht, die Farbe der Sättigungsleiste anzupassen.

## Features

- 🎨 Passe die Farbe der Sättigungsleiste nach deinen Wünschen an (RGB)
- ⚙️ GUI zur Verwaltung der Farbeinstellungen
- ⌨️ Anpassbare Tastenkombination (Standard: U)
- 💾 Einstellungen werden automatisch gespeichert
- 🌐 Unterstützung für Deutsch und Englisch

## Installation

1. Installiere Fabric Loader für Minecraft 1.21.11
2. Lade die MOD-Datei in dein `mods`-Verzeichnis
3. Starte Minecraft neu

## Verwendung

1. Drücke `U` im Spiel, um die Einstellungs-GUI zu öffnen
2. Gib die RGB-Werte für deine gewünschte Farbe ein (0-255)
3. Klicke auf "Speichern"
4. Die Farbe wird sofort angewendet und gespeichert

## Keybinds anpassen

1. Gehe zu Einstellungen → Steuerung
2. Suche nach "Sättigungs GUI öffnen"
3. Weise eine neue Taste zu

## Konfiguration

Die Einstellungen werden in `config/sattigungsmod.conf` gespeichert und sehen wie folgt aus:

```
red=255
green=0
blue=0
```
